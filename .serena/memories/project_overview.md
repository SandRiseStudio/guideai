# Project Overview

GuideAI is a Metacognitive Behavior Handbook Platform inspired by Meta AI's "Metacognitive Reuse" research. The goal is to productize the "behavior handbook" concept, allowing AI agents to discover, apply, and evolve reusable reasoning procedures ("behaviors") to reduce token usage and improve consistency.

## Core Components
- **CLI**: `guideai` command-line interface for managing behaviors, workflows, and actions.
- **REST API**: FastAPI-based endpoints for platform integration.
- **MCP Server**: Model Context Protocol implementation for exposing tools to AI assistants.
- **VS Code Extension**: IDE integration for behavior browsing, workflow execution, and compliance.

## Key Services
- **BehaviorService**: CRUD and versioning for behaviors.
- **WorkflowService**: Template management and behavior-conditioned inference.
- **ActionService**: Immutable action registry and replay.
- **RunService**: Execution tracking and progress monitoring.
- **ComplianceService**: Checklist automation.

## Success Metrics
- 70% behavior reuse rate.
- 30% token savings.
- 80% task completion rate.
- 95% compliance coverage.
