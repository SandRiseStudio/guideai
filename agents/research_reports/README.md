# AI Research Reports

This directory contains evaluation reports generated by the AI Research Agent.

## Purpose

Reports stored here are accessible to other agents for:
- **Planning**: Reference research findings when designing new features
- **Implementation**: Use recommended approaches and implementation roadmaps
- **Decision Making**: Check verdicts (ADOPT/ADAPT/DEFER/REJECT) for technology choices

## Report Format

Each report includes:
1. **Comprehension Summary** - Core idea, problem, solution, contributions
2. **Evaluation Scores** - Relevance, feasibility, novelty, ROI, safety
3. **Recommendation** - Verdict with implementation roadmap

## File Naming

Reports are named: `{title_slug}_{paper_id_prefix}.md`

Example: `agent_handbook_paper_fc.md`

## Usage by Other Agents

Other agents can reference these reports using semantic search or direct file access.
Reports are also stored in SQLite at `~/.guideai/research.db` for programmatic retrieval.
